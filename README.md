# Word Frequency Analyzer üìä

A modern web app to help language learners analyze books/documents, extract high-frequency vocabulary, and learn effectively.

## üöÄ Features & Roadmap

### Core Analysis & Visualization
- [x] **Multi-Format Support**: Upload `.pdf`, `.docx`, `.txt`, `.csv` (Max 100MB).
- [x] **Smart Analysis**: Extract top 500 frequent words, filter short words (3+ chars).
- [x] **Visualizations**: Interactive Word Cloud & detailed frequency statistics.
- [x] **Data Tools**: Search, filter, and sort word lists.
- [x] **Export**: Download analysis results to CSV.
- [x] **Vocabulary Management**: Star/Save words for review (Local).

### Learning Tools (Planned)
- [ ] **Context Viewer**: Show the original sentence/paragraph where the word appears in the book.
- [ ] **Instant Lookups**: Popup translation & auto-pronunciation (TTS) on hover/click.
- [ ] **Flashcards & SRS**: Built-in Spaced Repetition System for vocabulary review.
- [ ] **Anki Integration**: Export saved words to `.apkg` format.

### System & UX (Planned)
- [ ] **Mobile Optimization**: Complete responsive design for learning on the go.
- [ ] **User Accounts**: Secure login via Email OTP.
- [ ] **Cloud Sync**: Sync saved vocabulary and progress across devices.

### Community & Feedback (Planned)
- [ ] **Feedback Portal**: In-app form for users to submit ideas (supports rich text, images, video).
- [ ] **Admin Dashboard**: Internal tool for devs to view, label, and archive user feedback.
